# Инфраструктура локальной разработки

Сборник готовых рабочих конфигураций для локальной разработки.

## Сервисы

| Сервис                  | Описание                                                                     |
|-------------------------|------------------------------------------------------------------------------|
| [OpenTelemetry](otel)   | Набор сервисов для сбора телеметрии (Grafana, Loki, Tempo, Mimir, Collector) |
| [PostgresSQL](postgres) | База данных для приложений                                                   |


## Запуск

Создайте сеть для того что бы сервисы видели друг друга.

```bash
docker network create --attachable local-infra
```

Далее в зависимости от вашего стэка переходите в соответствующие папки и там выполняйте:

```bash
docker compose up -d
```
